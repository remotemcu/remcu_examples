STM32F4-Discovery Board Firmware Applications Package history

The STM32F4-Discovery Board Firmware Applications Package provides ready-to-run firmware examples to support quick evaluation and development on STM32F4-Discovery board. It contains the following examples:
  • Upgrading STM32F4DISCOVERY board firmware using a USB key, under Project\FW_upgrade folder, for more details about this application refer to the associated document AN3990
  • Audio playback and recording using the STM32F4DISCOVERY, under Project\Audio_playback_and_record, for more details about this application refer to the associated document AN3997
  • STM32F4DISCOVERY peripheral firmware examples, under Project\Peripheral_Examples, for more details about this application refer to the associated document AN3983
  • STM32F4DISCOVERY Board Demonstration firmware, under Project\Demonstration, for more details about this application and the overall package structure refer to the associated document "Getting started with software and firmware environments for the STM32F4DISCOVERY Kit (UM1467)"

For more information on the STM32F4-Discovery, to download the available firmware package and documentation visit www.st.com/stm32f4-discovery.
